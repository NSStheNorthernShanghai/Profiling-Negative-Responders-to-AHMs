# Profiling-Negative-Responders-to-AHMs

This repository contains the analysis code supporting the study:
**Profiling Negative Responders to Antihypertensive Medications Informs Prescription Practices: A Real-world Cohort Study**

## Data source
The data used in this study were obtained from the Chinese Cardiovascular Association (CCA) Database-Hypertension Center. Access to the study data was granted after review and approval by the relevant data access committee.
